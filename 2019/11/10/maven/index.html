maven 与 npm 对比

npm被标记为javascript的软件包管理工具，

maven被标记为java的构建自动化和依赖项管理工具。

pom.xml 与 package.json

  • package.json 只通过包名跟版本来确定(坑爹，各种抢占包名);
  • pom.xml 用 GAV (groupId, artifactId, version) 3个来判定

dependency:tree

  • maven 是扁平化的，相同的包只能有一个版本
  • npm 是 嵌套式，导致各种问题存在(有改善)，所以我现在基本用的yarn
